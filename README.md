# smart_incident_reporter
Smart Incident Reporter is a Flutter-based mobile application that allows authenticated users to create, manage, and track incident reports in a structured and user-friendly way. The application is built using Firebase as the backend, Riverpod for state management, and Beamer for navigation, following a clean MVVM architecture.

## Firebase Setup
a. Create a Firebase project
b. Enable:Firebase Authentication (Email/Password)
c. Cloud Firestore
d. Firebase Storage
e. Add app to Firebase
f. Download google-services.json and place it inside android/app.
